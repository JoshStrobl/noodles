## noodles lint

Runs available linters for projects

### Synopsis


Runs available linters for projects

```
noodles lint [flags]
```

### Options

```
  -c, --confidence float   Minimum confidence for linting problems (default 0.5)
  -h, --help               help for lint
  -p, --project string     Name of the project we're linting
```
